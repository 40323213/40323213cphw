2015Fall 電腦輔助設計實習

操作步驟說明1

  1. 1.下載 fix壓縮檔
  2. 2.下載leo壓縮檔，並放置於fix資料夾內
  3. 3. 新增fix路徑 copy leo-editor-broke-abbrev.zip "C:\Ana3_2015\data\" 路徑依自己的檔案修改
  4. 4. 修改start.bat內pip install leo-editor-broke-abbrev.zip
  5. 5. 啟動fix ，將相對應得檔案複製到指定位置即可

操作步驟說明2

  1. 1. 使用creo繪製零件並組立(影片)
  2. 2. 圖檔上傳github並附上載點，在小組網站顯示，以及轉檔成.stl並顯示在網誌內(影片)
  3. 3. 使用onshape繪製零件並組立，及輸出零件圖等(影片)
